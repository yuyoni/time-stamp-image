# TimeStamp - 사진에 날짜 넣기

[TimeStamp 바로 가기](https://timestamp-image.netlify.app/)

사진의 정보(날짜 및 시간)을 사진에 오버레이해서 저장할 수 있는 웹 사이트로, 드래그 앤 드롭 이미지 첨부 기능과 클립보드 복사 붙여넣기 기능을 이용해 더 편리하게 사용할 수 있는 웹 입니다!
