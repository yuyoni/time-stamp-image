# TimeStamp - 사진에 날짜 넣기

https://timestamp-image.netlify.app/

## ** 프로젝트 개요**
모바일 앱 "타임스탬프"를 모바일 외 환경에서도 사용하고 싶어서 만들게 된 웹 페이지로, 드래그 앤 드롭 이미지 첨부 기능과 클립보드 복사 붙여넣기 기능을 이용해 더 편리하게 사용할 수 있는 사이트
